# Мониторинг состояния каталогов отправки нормативно-справочной информации на сервере

**Название приложения:** postdirview.

**Назначение приложения:** мониторинг состояния каталогов отправки нормативно-справочной информации на предмет наличия в них файлов, подготовленных на отправку но не отправленных в отделения почтовой связи, а также отображение дополнительной иформации об этих файлах.

Структура каталогов в приложении отображается в виде дерева. Приложение используется инженерами группы технической поддержки для осуществления контроля получения рассылаемой нормативно-справочной информации.


Приложение написано на **Python 2.7** с использованием библиотеки кроссплатформенного графического интерфейса пользователя (GUI) **WxPython**.


Используются модули, не входящие в стандартную поставку интерпретатора:
        - wxPython.
